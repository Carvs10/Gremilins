1. Intorduction

The Gremlins is a memory manager which seeks to prioritize the allocation speed, for that for this purpose we made a class, which basically keeps the free spaces in a linked-list for a faster allocation. This project was developed following instructions and orientations given by Prof. Selan Rodrigues dos Santos. By building this, we want to see all the all applications of the subjects studied in EDB I.
